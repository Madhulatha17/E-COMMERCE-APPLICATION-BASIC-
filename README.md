# E-COMMERCE-APPLICATION-BASIC-

## Features

- Product Listings
- Product Details
- Shopping Cart
- Add To Cart
- Remove From Cart
- Entity Framework Integration

## Technologies

- ASP.NET MVC
- Entity Framework Core
- SQL Server
- .NET 8

## Run Project

```bash
dotnet restore
dotnet ef database update
dotnet run
```

## URL

https://localhost:5001
